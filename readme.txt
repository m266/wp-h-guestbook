Gästebuch-Plugin

== Description ==
Gästebuch auf Grundlage der Kommentarfunktion

== Installation ==
1. Upload 'wp-h-guestbook'  to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin in Dashboard -> WP H-Guestbook
